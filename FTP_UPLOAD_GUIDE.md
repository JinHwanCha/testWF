### 3. 업로드할 파일/폴더
로컬 `D:\testWF\` 폴더의 다음 파일들을 서버의 `/html/` 폴더에 업로드:

#### 필수 파일들:
```
admin/              (전체 폴더)
api/                (전체 폴더)
data/               (전체 폴더)
images/             (전체 폴더)
js/                 (전체 폴더)
css/                (전체 폴더)
work/               (전체 폴더)
news/               (전체 폴더)
people/             (전체 폴더)
index.html
```

#### 업로드 하지 않아도 되는 파일들:
```
node_modules/       (Node.js 패키지 - 필요 없음)
server.js           (Node.js 서버 - 필요 없음)
package.json        (Node.js 설정 - 필요 없음)
package-lock.json   (Node.js 설정 - 필요 없음)
```

### 4. 폴더 권한 설정 (중요!)

FTP 클라이언트에서 다음 폴더를 우클릭 → 파일 권한 → 755 또는 777로 설정:

- `/html/data/` → **777** (읽기/쓰기 필요)
- `/html/images/uploads/` → **777** (읽기/쓰기 필요)

> **권한 설정 방법 (FileZilla)**:
> 1. 폴더 우클릭
> 2. "파일 권한..." 선택
> 3. 숫자 값: 777 입력
> 4. "서브 디렉토리로 재귀" 체크
> 5. 확인

### 5. 업로드 완료 후 확인

브라우저에서 접속:
```
http://fisherman.dothome.co.kr/admin
```

## 🔧 권한 오류 해결

만약 "권한이 없습니다" 또는 "파일을 저장할 수 없습니다" 에러가 발생하면:

1. FTP로 접속
2. `/html/data/` 폴더 우클릭
3. 파일 권한: **777**
4. "서브 디렉토리로 재귀" 체크
5. 확인

같은 방법으로 `/html/images/uploads/` 폴더도 설정
